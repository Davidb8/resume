# Resume

Professional resume in LaTeX format.

## Building

Requires a LaTeX distribution (e.g., MacTeX, TeX Live, MiKTeX).

```bash
# Build PDF
make

# Clean auxiliary files
make clean

# Clean everything including PDF
make cleanall
```

## Editing

Edit `resume.tex` directly. The document uses standard LaTeX with these packages:
- `fontawesome5` for icons
- `hyperref` for clickable links
- `geometry` for margins
- `titlesec` for section formatting

## Online Editing

You can also import this into [Overleaf](https://www.overleaf.com) for online editing with real-time preview.
